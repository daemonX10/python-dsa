üéØ Sets

Set Creation and Characteristics

# Creating sets empty_set = set() # Note: {} creates an empty dict, not set numbers = {1, 2, 3, 4, 5} from_list = set([1, 2, 3, 2, 1]) # {1, 2, 3} - duplicates removed from_string = set("hello") # {'h', 'e', 'l', 'o'} # Set comprehension squares = {x**2 for x in range(1, 6)} # {1, 4, 9, 16, 25} evens = {x for x in range(10) if x % 2 == 0} # {0, 2, 4, 6, 8}
üéØ Unique Elements: Sets automatically remove duplicates and store only unique elements. Elements must be hashable (immutable).

Essential Set Operations

s.add(item) O(1) avg

Add single element

s = {1, 2, 3} s.add(4) # {1, 2, 3, 4} s.add(2) # {1, 2, 3, 4} - no change

s.update(iterable) O(k)

Add multiple elements

s = {1, 2, 3} s.update([4, 5, 6]) # {1, 2, 3, 4, 5, 6} s.update("abc") # Adds 'a', 'b', 'c'

s.remove(item) O(1) avg

Remove element (raises KeyError if not found)

s = {1, 2, 3, 4} s.remove(3) # {1, 2, 4} # s.remove(5) # KeyError!

s.discard(item) O(1) avg

Remove element (no error if not found)

s = {1, 2, 3, 4} s.discard(3) # {1, 2, 4} s.discard(5) # No error, set unchanged

s.pop() O(1)

Remove and return arbitrary element

s = {1, 2, 3} item = s.pop() # Returns and removes random element # Empty set: s.pop() raises KeyError

s.clear() O(1)

Remove all elements

s = {1, 2, 3, 4, 5} s.clear() # set()

len(s) O(1)

Get number of elements

s = {1, 2, 3, 4, 5} size = len(s) # 5

item in s O(1) avg

Check membership (very fast!)

s = {1, 2, 3, 4, 5} has_3 = 3 in s # True has_6 = 6 in s # False

Set Mathematical Operations

set1 = {1, 2, 3, 4, 5} set2 = {4, 5, 6, 7, 8} # Union (combine all unique elements) union1 = set1 | set2 # {1, 2, 3, 4, 5, 6, 7, 8} union2 = set1.union(set2) # Same as above # Intersection (common elements) intersection1 = set1 & set2 # {4, 5} intersection2 = set1.intersection(set2) # Same as above # Difference (elements in first but not second) difference1 = set1 - set2 # {1, 2, 3} difference2 = set1.difference(set2) # Same as above # Symmetric difference (elements in either but not both) sym_diff1 = set1 ^ set2 # {1, 2, 3, 6, 7, 8} sym_diff2 = set1.symmetric_difference(set2) # Same as above

Set Comparison Operations

set1 = {1, 2, 3} set2 = {1, 2, 3, 4, 5} set3 = {1, 2, 3} # Subset checks is_subset1 = set1 <= set2 # True (set1 is subset of set2) is_subset2 = set1.issubset(set2) # True (same as above) # Superset checks is_superset1 = set2 >= set1 # True (set2 is superset of set1) is_superset2 = set2.issuperset(set1) # True (same as above) # Disjoint check (no common elements) set4 = {6, 7, 8} are_disjoint = set1.isdisjoint(set4) # True # Equality are_equal = set1 == set3 # True

Common Set Patterns

Remove Duplicates from List

# Quick way to remove duplicates (loses order) numbers = [1, 2, 3, 2, 4, 1, 5, 3] unique_numbers = list(set(numbers)) # Order may change # Preserve order while removing duplicates def remove_duplicates_ordered(lst): seen = set() result = [] for item in lst: if item not in seen: seen.add(item) result.append(item) return result ordered_unique = remove_duplicates_ordered(numbers) # [1, 2, 3, 4, 5]

Find Common Elements

# Find common elements between multiple lists list1 = [1, 2, 3, 4, 5] list2 = [4, 5, 6, 7, 8] list3 = [5, 6, 7, 8, 9] common = set(list1) & set(list2) & set(list3) # {5} # Find elements present in any list all_elements = set(list1) | set(list2) | set(list3); # Find elements unique to first list unique_to_first = set(list1) - set(list2) - set(list3);

Efficient Membership Testing

# Instead of checking membership in list (O(n)) vowels_list = ['a', 'e', 'i', 'o', 'u'] def is_vowel_slow(char): return char.lower() in vowels_list # O(n) for each check # Use set for O(1) membership testing vowels_set = {'a', 'e', 'i', 'o', 'u'} def is_vowel_fast(char): return char.lower() in vowels_set # O(1) for each check # Count vowels in text text = "Hello World" vowel_count = sum(1 for char in text if is_vowel_fast(char))

Set-based Algorithms

# Check if two strings are anagrams def are_anagrams_set(s1, s2): return set(s1.lower()) == set(s2.lower()) # Find all unique characters in text def unique_chars(text): return set(text.lower().replace(" ", "")) # Check if string has all unique characters def has_unique_chars(s): return len(s) == len(set(s)) # Find missing numbers in sequence def find_missing_numbers(nums, start, end): expected = set(range(start, end + 1)) actual = set(nums) return sorted(expected - actual) missing = find_missing_numbers([1, 3, 5, 7], 1, 10) # [2, 4, 6, 8, 9, 10]
üí° Pro Tips:
  • Sets provide O(1) average-case membership testing
  • Use sets when you need unique elements and fast lookups
  • Set operations (union, intersection) are very efficient
  • Only hashable (immutable) objects can be set elements
üöÄ Performance Tips:
  • Convert lists to sets for fast membership testing
  • Use set comprehensions for filtering unique elements
  • Set operations are implemented in C and very fast
  • frozenset for immutable sets that can be set elements
‚ö†Ô∏è Set Limitations:
  • Sets are unordered (before Python 3.7) or insertion-ordered
  • Cannot contain mutable objects (lists, dicts, sets)
  • No indexing - can't access elements by position
  • Use frozenset if you need a hashable set
