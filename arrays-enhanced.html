Essential List Methods

append(item) O(1)

Add item to end of list

lst = [1, 2, 3] lst.append(4) # [1, 2, 3, 4]

insert(index, item) O(n)

Insert item at specific index

lst = [1, 2, 3] lst.insert(1, 'new') # [1, 'new', 2, 3]

remove(item) O(n)

Remove first occurrence of item

lst = [1, 2, 3, 2] lst.remove(2) # [1, 3, 2]

pop(index) O(1) end, O(n) middle

Remove and return item at index

lst = [1, 2, 3] item = lst.pop() # item = 3, lst = [1, 2] item = lst.pop(0) # item = 1, lst = [2]

extend(iterable) O(k)

Add all items from iterable

lst1 = [1, 2, 3] lst2 = [4, 5, 6] lst1.extend(lst2) # [1, 2, 3, 4, 5, 6]

sort() O(n log n)

Sort list in place

lst = [3, 1, 4, 1, 5] lst.sort() # [1, 1, 3, 4, 5] lst.sort(reverse=True) # [5, 4, 3, 1, 1]

🎯 Two Pointers Technique

Use two pointers moving from opposite ends to solve problems efficiently.

def two_sum(nums, target): """Find two numbers that add up to target""" left, right = 0, len(nums) - 1 while left < right: current_sum = nums[left] + nums[right] if current_sum == target: return [left, right] elif current_sum < target: left += 1 # Need larger sum else: right -= 1 # Need smaller sum return []

def max_sum_subarray(nums, k): """Find maximum sum of k consecutive elements""" if len(nums) < k: return 0 # Calculate sum of first window window_sum = sum(nums[:k]) max_sum = window_sum # Slide the window for i in range(k, len(nums)): # Remove leftmost, add rightmost window_sum = window_sum - nums[i - k] + nums[i] max_sum = max(max_sum, window_sum) return max_sum
💡 Pro Tips:
  • Use list comprehensions for readable code
  • Remember that lists are mutable and passed by reference
  • Use enumerate() when you need both index and value
  • Use zip() to iterate over multiple lists simultaneously
🚀 Performance Tips:
  • append() is O(1), insert(0, item) is O(n)
  • Use collections.deque for frequent insertions at both ends
  • List comprehensions are generally faster than for loops
  • Use set() for O(1) membership testing instead of "in" on lists
