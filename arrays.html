Python offers multiple ways to create and initialize lists. Each method has its use cases:

# Creating lists empty_list = [] # Empty list: [] numbers = [1, 2, 3, 4, 5] # List with numbers mixed = [1, "hello", 3.14, True] # Mixed types allowed from_range = list(range(10)) # [0, 1, 2, ..., 9] # List comprehensions - Pythonic way! squares = [x ** 2 for x in range(10)] # [0, 1, 4, 9, 16, ...] evens = [x for x in range(20) if x % 2 == 0] # [0, 2, 4, 6, ...] nested = [[i * j for j in range(3)] for i in range(3)] # 2D list

Essential List Methods

append(item) O(1)

Add item to end of list

lst = [1, 2, 3] lst.append(4) # [1, 2, 3, 4]

insert(index, item) O(n)

Insert item at specific index

lst = [1, 2, 3] lst.insert(1, 'new') # [1, 'new', 2, 3]

remove(item) O(n)

Remove first occurrence of item

lst = [1, 2, 3, 2] lst.remove(2) # [1, 3, 2]

pop(index) O(1) end, O(n) middle

Remove and return item at index

lst = [1, 2, 3] item = lst.pop() # item = 3, lst = [1, 2] item = lst.pop(0) # item = 1, lst = [2]

extend(iterable) O(k)

Add all items from iterable

lst1 = [1, 2, 3] lst2 = [4, 5, 6] lst1.extend(lst2) # [1, 2, 3, 4, 5, 6]

sort() O(n log n)

Sort list in place

lst = [3, 1, 4, 1, 5] lst.sort() # [1, 1, 3, 4, 5] lst.sort(reverse=True) # [5, 4, 3, 1, 1]
