Essential List Methods

append(item) O(1)

Add item to end of list

lst = [1, 2, 3] lst.append(4) # [1, 2, 3, 4]

insert(index, item) O(n)

Insert item at specific index

lst = [1, 2, 3] lst.insert(1, 'new') # [1, 'new', 2, 3]

remove(item) O(n)

Remove first occurrence of item

lst = [1, 2, 3, 2] lst.remove(2) # [1, 3, 2]

pop(index) O(1) end, O(n) middle

Remove and return item at index

lst = [1, 2, 3] item = lst.pop() # item = 3, lst = [1, 2] item = lst.pop(0) # item = 1, lst = [2]

extend(iterable) O(k)

Add all items from iterable

lst1 = [1, 2, 3] lst2 = [4, 5, 6] lst1.extend(lst2) # [1, 2, 3, 4, 5, 6]

sort() O(n log n)

Sort list in place

lst = [3, 1, 4, 1, 5] lst.sort() # [1, 1, 3, 4, 5] lst.sort(reverse=True) # [5, 4, 3, 1, 1]

🧠Common Patterns & Algorithms

🔄 Two Pointers Technique

Use two pointers moving towards each other or in the same direction to solve problems efficiently.

# Example: Check if array is palindrome def is_palindrome(arr): left, right = 0, len(arr) - 1 while left < right: if arr[left] != arr[right]: return False left += 1 right -= 1 return True

🪟 Sliding Window Technique

Maintain a "window" of elements and slide it through the array to find optimal solutions.

# Example: Maximum sum of k consecutive elements def max_sum_k_elements(arr, k): window_sum = sum(arr[:k]) # First window max_sum = window_sum for i in range(k, len(arr)): window_sum = window_sum - arr[i - k] + arr[i] max_sum = max(max_sum, window_sum) return max_sum

🎯 Binary Search

Efficiently search in sorted arrays by repeatedly dividing the search space in half.

# Binary search implementation def binary_search(arr, target): left, right = 0, len(arr) - 1 while left <= right: mid = (left + right) // 2 if arr[mid] == target: return mid elif arr[mid] < target: left = mid + 1 else: right = mid - 1 return -1 # Not found

📊 Sorting Algorithms

Different algorithms for arranging elements in order.

Bubble Sort O(n²)
def bubble_sort(arr): n = len(arr) for i in range(n): for j in range(0, n - i - 1): if arr[j] > arr[j + 1]: arr[j], arr[j + 1] = arr[j + 1], arr[j]
Quick Sort O(n log n)
def quick_sort(arr): if len(arr) <= 1: return arr pivot = arr[len(arr) // 2] left = [x for x in arr if x < pivot] middle = [x for x in arr if x == pivot] right = [x for x in arr if x > pivot] return quick_sort(left) + middle + quick_sort(right)
